		int i1 = 6;
		System.out.println("i\: " + i1);
		
		System.out.println(s1.toString())		// Invia un messaggio all'oggetto s1 col metodo toString, il quale stampa a video la stringa. Se la stringa è vuota, il compiler si lagna.

--- Altra sintassi
Identificatori: nomi assegnati ai vari dati come variabili o costanti. No, non possono essere keywords del linguaggio.
	- nomi composti, camelcase
	- classnames devono avere la prima lettera maiuscola
	- metodi, oggetti e campi devono iniziare in lowecase.
	- Le costanti vanno in caps lock. Con nomi composti, usare '_'.

-- Espressioni
Termine eventualmente valutato ad un valore. Le espressioni hanno un tipo associato.
eval(4) = 4
eval(x+1) = eval(x) + eval(1) = 3+1 = 4.
Esistono espressioni booleane, aritmetiche e di riferimento, queste ultime se valutano ad un qualunque oggetto.

Operatore ternario: expr1 ? val1 : val2		// expr1 è vera? Sì=val1, No=val2.

-- Costrutti di controllo flow
Puoi dominare lo scope attraverso le {}.
{
	// Scope1
} {
	// Scope2
}
Così sono due scope diversi, tuttavia se ne dichiari uno innestato, saprà leggere anche il suo sovrascope.
{
	// Scope 1
	{
		// Scope 2, ma legge ancora anche su scope 1
	}
}
