--- Tipi e oggetti ---
Linguaggi tipati; che hanno un tipo come int, char, etc. Lo definiamo un insieme di valori con relative operazioni effettuabili su di essi.

Se le variabili a runtime non possono assumere tipi diversi, si dicono fortemente tipizzati

Tipi primitivi di java:
	- int, interi da -2^{31} a 2^{31}-1, operazioni: +-*/%
	- double
	- char, non hanno operazioni
	- boolean, operazioni !&&||

Type casting ha sintassi uguale a C, per passare per esempio da float a int di fa tipo int i = (int) 3.2;

Esiste il tipo stringa ma sono degli oggetti per renderle un attimo meno problematiche. Puoi concatenarle con +.

-- Oggetti
Istanze di classi accessibili solamente tramite messaggi. Hanno dei campi, sottoinsiemi di locazioni di memoria, interagibili coi metodi.
Java ha puntatori ma non ne esiste l'aritmetica.

Referenza dell'oggetto contiene il nome di una classe ed un indirizzo di memoria. Non usa memoria, tiene solo l'indirizzo.

public class Vehicle {				// Rivedere la definizione di classe, potrei aver scritto na minchiata. Era una minchiata.
	string Manifacturer;
	setFrameNumber(int);
};
Vehicle MyCar; 				// Contiene Vehicle@null
MyCar = new Vehicle();		// MyCar ora è un oggetto di classe Vehicle. Istanziazione. Il compiler vedrà Vehicle@memAddress
MyCar.setFrameNumber(2);	// Messaggio legale inviato alla zona di memoria memAddress.
